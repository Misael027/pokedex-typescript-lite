# Pokédex TypeScript Lite

Uma aplicação interativa de terminal desenvolvida em TypeScript e Node.js que consome a [PokeAPI](https://pokeapi.co/) para buscar dados de Pokémon em tempo real, permitindo gerenciar um catálogo/mochila pessoal com regras de negócio específicas.

# Funcionalidades

- Busca em Tempo Real: Conexão com a API pública PokeAPI para extrair nome, ID, tipos, peso e altura do Pokémon.
- Gerenciamento de Catálogo: Adicione e remova Pokémon da sua mochila através de um menu interativo.
- Validação de Duplicidade: O sistema impede a adição de Pokémon repetidos no catálogo.
- Tratamento de Erros: Exibe mensagens amigáveis caso o Pokémon digitado não seja encontrado ou ocorra falha de conexão.
- Interface Interativa: Menu via terminal controlado pelo teclado do usuário.
